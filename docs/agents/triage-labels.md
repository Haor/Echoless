# 分诊标签

Echoless 使用以下标签词汇：

| 标签 | 含义 |
|---|---|
| `needs-triage` | 任务单尚未完成分类或有效性确认。 |
| `needs-info` | 继续推进需要补充复现信息、日志、决策或环境资料。 |
| `ready-for-agent` | 任务单已充分定义，可以由自动代理自主执行。 |
| `ready-for-human` | 下一步需要人工判断、凭据、硬件、安装包运行验证或用户验收。 |
| `wontfix` | 任务单已经理解，但被明确排除在产品范围之外。 |

同一时间只使用一个就绪标签。任务单状态变化时移除已经失效的标签。
